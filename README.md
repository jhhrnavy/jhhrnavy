## :pushpin: Intro
게임 콘텐츠 구현과 깔끔한 게임 구조 설계에 열정을 가진 유니티 클라이언트 개발자입니다.    
새로운 지식을 탐구하고 활용하는 것을 즐기며, Unity 엔진을 활용하여 사용자에게 최상의 경험을 제공하고자 합니다.    
게임 개발 시의 게임 구조, 코드의 가독성과 재사용성을 높이는 것에 중점을 두었습니다.     
질문과 피드백은 언제나 환영합니다.
</br>
